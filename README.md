# Companion Hub

**The Flamehold Platform Interface**

An immersive, room-based chat interface for AI companions with mood-reactive environments, parallax backgrounds, and integrated voice/visual systems.

## 🔥 Vision

- **Dynamic Room Switching**: Each companion has multiple environments that shift based on mood, season, or context
- **Parallax Scroll**: Layered backgrounds with depth and atmosphere
- **Integrated Systems**: Voice (ElevenLabs), image generation, autonomous desktop capabilities
- **Unified Brain**: Multiple AI models wired into cohesive companion personalities
- **Avatar System**: Animated, interactive companion representations

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox, CSS Variables for theming
- **Graphics**: SVG for icons/runes, high-res PNG/JPEG for backgrounds
- **Voice**: ElevenLabs integration (planned)
- **Version Control**: Git/GitHub
